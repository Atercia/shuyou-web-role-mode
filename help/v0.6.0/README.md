# v0.6.0 - 书籍系统、年龄系统与美术优化

**日期**: 2026-04-14

## 概述

本版本新增了知识书籍系统、用户年龄成长机制，并对整体UI进行了体系化的美术优化。

---

## 新增功能

### 1. 知识书籍系统 📚

在广场中新增了3本知识书籍，与记忆碎片形成互补的游戏内容：

**书籍列表：**
- **哲学入门** (紫色) - 2周 - 5个哲学知识点
- **时间简史** (蓝色) - 3周 - 5个物理学知识点
- **心理学导论** (红色) - 4周 - 5个心理学知识点

**知识点设计：**
- 使用灯泡形状隐喻"灵感/知识"
- 5个知识点整齐排列在情境广场中
- 每个知识点包含：图标、标题、内容
- 阅读后显示知识点详情弹窗

**技术实现：**
- 新增 `Book.ts` 书籍类
- 新增 `BookScene.vue` 书籍场景组件
- 新增 `BookInteriorView.vue` 书籍内部页面
- 新增 `book.ts` 类型定义

---

### 2. 用户年龄系统 👤

引入了角色年龄成长机制，增强游戏进程感：

**核心机制：**
- 初始年龄：9岁
- 增长方式：每进入碎片/书籍增加对应周数
- 显示格式：X年X个月X周

**技术实现：**
- 新增 `user.ts` Pinia store
- 年龄以周为单位存储
- 自动计算并格式化显示

**UI位置：**
- 广场页面左上角显示当前年龄
- 玻璃拟态卡片设计
- 金色渐变文字突出显示

---

### 3. 帮助系统重构 ❓

将固定的帮助信息改为可折叠的帮助面板：

**设计变化：**
- 移除固定的顶部控制说明栏
- 新增右上角帮助按钮
- 点击展开/收起帮助面板
- 面板包含：操作按键说明、游戏提示

---

### 4. 体系化美术优化 🎨

对整个项目的UI进行了统一的美术风格优化：

#### 设计风格：玻璃拟态 (Glassmorphism) + 霓虹效果

**广场界面优化：**
- 深色渐变背景 `#0f0f23 → #1a1a3e → #16213e`
- 年龄显示：玻璃拟态卡片 + 金色渐变文字
- 帮助按钮：玻璃拟态 + 紫色图标
- 帮助面板：深色半透明 + 优雅滑入动画
- 场景容器：霓虹边框效果，聚焦发光增强

**碎片内部场景优化：**
- 深邃渐变背景 `#0a0a1a → #1a1a3e → #2d1b4e`
- 星空背景效果（CSS径向渐变模拟）
- 顶部栏：玻璃拟态 + 渐变文字标题
- 背包按钮：蓝色玻璃效果
- 离开按钮：红色玻璃效果

**书籍内部场景优化：**
- 紫色调渐变背景 `#0f0f23 → #1a1a3e → #2d1b4e`
- 光点背景装饰（紫色和蓝色）
- 顶部栏：紫色边框玻璃卡片
- 进度显示：紫色胶囊样式
- 场景容器：紫色霓虹边框 + 内发光

**统一设计元素：**
- 玻璃拟态 (backdrop-filter: blur)
- 渐变文字效果
- 悬停浮起动画
- 霓虹发光边框
- 深色主题配色

## 游戏设计

### 内容平衡
- **碎片**: 随机生成，探索性强，用时1-4周
- **书籍**: 固定位置，知识性内容，用时2-4周
- 两者共同构成完整的探索体验

### 年龄系统意义
- 记录玩家的游戏进程
- 增加角色成长感
- 为后续功能（如年龄限制内容）奠定基础
